GOTI 22 Popup

Adicione um menu flutuante futurista às suas páginas com o GOTI 22! Com visual Chroma, botões dark, e scripts como SPeak, Khan Academy e Expansão Noturna, é perfeito para computador e celular.

Arrastar para Favoritos (PC)

Instruções

Computador (Chrome): Arraste o botão "Arrastar para Favoritos" para a barra de favoritos. Clique no favorito em qualquer página para abrir o popup. Use Shift direito para ocultar/exibir.

Celular (Chrome Mobile): Clique em "Copiar Bookmarklet", crie um favorito, e cole o código no campo URL. Toque no favorito para abrir o popup. Nota: Pode ser necessário usar o console remoto (veja suporte). Para melhor experiência, contate o suporte para a versão mobile.

Escolas: Se o bookmarklet não funcionar, cole o código no console (Ctrl+Shift+I). Se as URLs forem bloqueadas, contate o suporte.
